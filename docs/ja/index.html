多変量正規分布

このプログラムは多変量正規分布の計算をする。 Zdravko I. Botev 氏のRパッケージTruncatedNormal のC++版である。

  • GitHubソースファイル

インストール

./configure --prefix=YOUR_INSTALL_DIR
make
make install

準備

環境変数 DIGITAL_NET_PATH をインストールディレクトリ以下のshare/truncatednormalに セットする。

export DIGITAL_NET_PATH=YOUR_INSTALL_DIR/share/truncatednormal

LIBRARY_PATH と CPLUS_INCLUDE_PATH を適切にセットする。

注意

データファイル 'nxlw.dat' と 'solw.dat' はLittel Endianシステムで作られた バイナリファイルであり、 Big Endian システムでは正確に動作しないだろう。

使用法

多変量正規分布の計算をするC++関数はMCQMCIntegration::truncatedNormal()である。 この関数の引数としてMCQMCIntegration::DigitalNetIDを使用する。 この関数の返却値はMCQMCIntegreation::TruncatedNormalResultである。

MCQMCIntegration::truncatedNormal()の引数

s
s 多変量正規分布の次元である。直接の引数ではないが, 以下の引数に関わるもの。4 <= s <= 10
const std::vector< double > & lower
積分領域の下限を表すベクトル。長さは積分領域の次元sである。
const std::vector< double > & upper
積分領域の上限を表すベクトル。長さは積分領域の次元sである。
const std::vector< std::vector< double > > & sigma
分散・共分散行列。s次正方行列。
uint64_t number
積分を行う回数。この回数だけ積分を繰り返し、平均と誤差を求める。
DigitalNetID dnid
使用するデジタルネットを表す。
  • NXLW : WAFOM値の低い Niederreiter-Xing 点集合。
  • SOLW : WAFOM値の低い　Sobol 点集合。
uint32_t m
デジタルネットの上位mビットまでの精度, また１回の積分で2^m個の点を使用する。10 <= m <= 18。
double probability
積分結果が誤差内に入ると期待される確率。0.95, 0.99, 0.999, 0.9999のいずれか。

MCQMCIntegration::truncatedNormal()の返却値

probability
指定された正規分布の値(累積確率)
absoluteError
絶対誤差
relativeError
相対誤差
upperBound
正規分布の値の理論上の上限(Botevによる)
success
計算が成功したかどうかを示す。true:成功, false:失敗
