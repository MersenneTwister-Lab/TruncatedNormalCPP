compute Multivariate Normal Distribution by Quasi Monte-Carlo method.

computes an estimator and a deterministic upper bound of the probability P r(lower < X < upper), where X is a zero-mean multivariate normal vector with covariance matrix Σ, that is, X is drawn from N(0, Σ) infinite values for vectors upper and lower are accepted.

Note
parameter dn, digital net should have dimension s = lower.size - 1. And s should be an integer 4 <= s <= 10, therefore, lower.size size should be 5 <= size <= 11. Moreover, The F2-dimension of element of digitalnet, m should be 10 <= m <= 18.
probability should be one of {0.95, 0.99, 0.999, 0.9999}
Parameters
[in]lowerlower truncation limit.
[in]upperupper truncation limit.
[in]sigmacovariance matrix of N(0, Σ)
[in]numberMonte Carlo simulation effort — the larger the n, the smaller the relative error of the estimator.
[in,out]dndigital net.
[in]probabilityexpected probability such that the result is in the range [p-σ, p+σ].
Returns
estimated value of probability and other information.

compute Multivariate Normal Distribution by Monte-Carlo method.

computes an estimator and a deterministic upper bound of the probability P r(lower < X < upper), where X is a zero-mean multivariate normal vector with covariance matrix Σ, that is, X is drawn from N(0, Σ) infinite values for vectors upper and lower are accepted; Monte Carlo method uses sample size number;

dnid is one of

  • DigitalNet::digital_net_id::NXLW
  • DigitalNet::digital_net_id::SOLW Sobol Low WAFOM
  • DigitalNet::digital_net_id::RANDOM Monte-Carlo method

probability should be one of {0.95, 0.99, 0.999, 0.9999}

[in]upperupper truncation limit.
[in]sigmacovariance matrix of N(0, Σ)
[in]trialNumberMonte Carlo simulation effort — the larger the number, the smaller the relative error of the estimator.

estimated value of probability and other information.
